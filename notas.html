Introducción al CSS.

Como se escribe un comentario de CSS/*no se codifica */.
Como se escribe en el CSS un selector, que toma un componente del HTML dándole atributos visuales y de posición:
.selector{
propiedad: valor;
}

Ejemplo:

.card__title,
.card__subtitle {
font-weight: 700;
line-height: 1.2;
margin: 0 0 1rem;
}
Allí se muestra como se pueden sumar selectores por medio del uso de la "," el uso de guión reemplaza el espaciado ya que su uso implica a que el CSS tome lo escrito a la derecha del espaciado como un selector más específico.
Sobre la especificidad del CSS, la cascada de estilo permite que atributos de un elemento se heredadan a los dependientes, es decir, los que se escriben despúes. Si el selector de "body" tiene un color verde de la fuente y un selector h1 tiene un color amarillo, se codificara en el h1 el color amarillo, al ser más específico.

Publicado el 20 de agosto de 2021
Autor: David S. Silva Bernales

HTML: HyperText Markup Language/Lenguaje de Marcado para Hipertextos.
Es un lenguaje que utiliza un sistema de etiquetas o corchetes y contenido: 

<address><Matta 12, Viña del Mar</address>

Esas etiquetas pueden incluir otras etiquetas, lo que se denomina anidación. Y para poder tener un orden se recomienda identar, para que visualmente sea más fácil de leer y revisar, todo lo escrito debe tener cuidado del uso de mayúsculas, ocupar comillas dobles, tener cuidado con tener cerradas las etiquetas.

La estructura del HTML se compone por head que tienen los metadatos, body que es lo qel usuario ve e interactúa, existen un gran universo de etiquetas que están más detalladas en las presentaciones y links recomendados de los docentes.

Las etiquetas con su nombre y propiedades jerarquizan y estructuran el contenido de la página: div, span, header, nav, aside, article, footer, entre otras.
